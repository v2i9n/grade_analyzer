# 📊 Grade Chart - محلل الدرجات بالرسم البياني

**مشروعي الثاني بلغة بايثون** 💪  
برنامج تفاعلي يحلل درجاتك ويرسم لك رسم بياني ملون احترافي + يحسب معدلك تلقائياً

### ✨ المميزات
- إدخال عدد لا نهائي من المواد والدرجات
- تلوين تلقائي للأعمدة حسب الدرجة:
    - 🟢 أخضر: ممتاز 90+
    - 🔵 أزرق: جيد جداً 80+  
    - 🟠 برتقالي: جيد 70+
    - 🔴 أحمر: يحتاج تركيز
- حساب المعدل العام + خط أحمر يوضح معدلك على الرسم
- دعم كامل للغة العربية
- واجهة سهلة لأي طالب
